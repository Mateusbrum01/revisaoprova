Revisão para a prova

HTML



O HTML funciona por meio de tags, como p, para parágrafos, h1 até h6, para títulos, a, para links, img, para imagens e ul ol, li, para listas. A estrutura de uma página HTML começa com a tag !DOCTYPE html e depois segue com html, head (onde vão as informações da página) e body (onde vai o conteúdo visível). Além disso, é possível criar formulários com form, input, textarea e botões. Para organizar dados, usamos tabelas com table, tr, td e th.

CSS

O CSS (Cascading Style Sheets) serve para estilizar o conteúdo de páginas web, separando a estrutura (HTML) da apresentação visual. Com o CSS, é possível definir cores, fontes, margens, espaçamentos, posicionamento e até animações, proporcionando um design mais atrativo e funcional. Principais tags:
style: Usada para incluir CSS diretamente no HTML. body: Define o estilo do corpo da página. div: Contêiner genérico para estruturar a página. .classe: Seletor de classe para aplicar estilos a elementos com a mesma classe. #id: Seletor de ID para estilizar um único elemento com ID específico.

GIT



O Git é um sistema de controle de versão distribuído usado para gerenciar alterações no código de um projeto. Ele facilita a colaboração entre desenvolvedores, permitindo que trabalhem no mesmo projeto sem sobrescrever o trabalho uns dos outros, além de manter um histórico de alterações que pode ser revertido. Principais comandos:
git init: Inicializa um repositório Git. git add: Adiciona alterações ao estágio de preparação. git commit: Registra as alterações. git push: Envia alterações para o repositório remoto. git pull: Atualiza o repositório local.

GITHUB

O GitHub é uma plataforma de hospedagem de código-fonte baseada no Git, que permite armazenar repositórios de projetos, colaborar com outros desenvolvedores e compartilhar código. É amplamente utilizado para projetos open source e privados. Principais funcionalidades:
Repositórios: Espaços para armazenar e gerenciar o código. Fork: Criação de uma cópia de um repositório para trabalhar de forma independente. Pull Requests: Solicitação de inclusão de mudanças em um repositório principal. Issues: Ferramenta para gerenciar tarefas, bugs ou melhorias.

JAVASCRIPT

O JavaScript é uma linguagem de programação usada principalmente para criar interatividade em páginas web. Ele permite que o conteúdo de uma página HTML seja dinâmico, respondendo a ações do usuário, como cliques, rolagem e digitação, sem precisar recarregar a página. Principais recursos:
Manipulação do DOM: Permite acessar e modificar elementos HTML em tempo real. Eventos: Lida com ações do usuário, como cliques, teclas pressionadas, etc. AJAX: Realiza requisições assíncronas ao servidor para carregar dados sem atualizar a página. Funções: Blocos de código reutilizáveis para organizar e modularizar o código.
